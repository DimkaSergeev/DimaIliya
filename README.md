## ОПИСАИЕ ПРОЕКТА 
Вся доставка - это сервис срочной курьерской доставки по городу. В этом проекте требовалось освежить дизайн, но при этом основные моменты и контент оставить, как на старом сайте. Важно подчеркнуть современность и удобство для пользователей.
### Технологии в проекте:
* UX/UI
* Figma
* HTML/CSS/JS
* VS code
* GitHub
* WordPress
