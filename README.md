## ОПИСАИЕ ПРОЕКТА 
Вся доставка - это сервис срочной курьерской доставки по городу. 
В данном проекте требуется освежить дизайн, но при этом основные моменты и контент оставить, как на старом сайте. Важно подчеркнуть современность и удобство для пользователей.
### Технологии в проекте:
* UX/UI;
* Figma;
* HTML/CSS/JS;
* VS code;
* GitHub;
* WordPress.
## СПЕЦИФИЧЕСКИЕ ОСОБЕННОСТИ ПРОЕКТА 
### Проблемы:
1. Сложность в натяжке на CMS WordPress;
2. Сложность реализации входа в личный кабинет для физических лиц, интернет-магазинов и партнеров компании. 
## ТЕХНИЧЕСКОЕ ОПИСАНИЕ ПРОЕКТА, РУКОВОДСТВО
### Техническое описание
Это веб-сайт, переход на который будет осуществлен с помощью компьютера и мобильных устройств через веб-браузер.
