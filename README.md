## ОПИСАИЕ ПРОЕКТА 
Вся доставка - это сервис срочной курьерской доставки по городу. В данном проекте требуется освежить дизайн, но при этом основные моменты и контент оставить, как на старом сайте. Важно подчеркнуть современность и удобство для пользователей.
### Технологии в проекте:
* UX/UI;
* Figma;
* HTML/CSS/JS;
* VS code;
* GitHub;
* WordPress.
## СПЕЦИФИЧЕСКИЕ ОСОБЕННОСТИ ПРОЕКТА 
### Проблемы:
1. С натяжкой на CMS WordPress;
2. С реализацией входа в личный кабинет для физических лиц, интернет-магазинов и партнеров компании. 
## ТЕХНИЧЕСКОЕ ОПИСАНИЕ ПРОЕКТА, РУКОВОДСТВО
Это веб-сайт, переход на который осуществляется в веб-браузере с помощью компьютера или мобильных устройств. Данный сайт предоставляет возможность оформления доставки для бизнеса и поиска заказов. Услуги доставки предлагаются для физических лиц, интернет-магаинов и партнеров компании. Также имеется возможность просмотра новостей и различных тарифов доставки.
